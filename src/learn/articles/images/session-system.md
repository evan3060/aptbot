设计一张现代企业级中文架构图信息图。主题：AI Agent Session 系统——JSONL 持久化与多用户隔离。整体布局：左侧多个用户客户端（UserA/UserB/UserC）各自连接到自己 Session。每个 Session 包含主文件 jsonl append-only 和 sidecar meta.json。右侧两级缓存：per-sessionKey ring buffer 1000 + 全局 LRU 50000。底部 UserStorage + scrypt + Bearer token 多用户隔离。整体风格：Figma UI 风格，蓝白科技风，扁平化矢量图标，细边框卡片，微阴影，浅灰背景，科技感网格纹理。配色：白色背景，蓝色主色调，绿色代表活跃 session，灰色代表历史 session，深蓝文字。从左到右分层布局，蓝色箭头连接。画面质量：超高分辨率，专业 PPT 级视觉。size: 1024x1024
